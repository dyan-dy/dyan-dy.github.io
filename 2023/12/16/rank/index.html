Rank correlation
更新于 2023-12-16

In Spearman correlation, we consider a nonparametric measure of rank correlation, which is, statistical dependence between the rankings of two variables. Unlike Pearson correlation, Spearman correlation does not assume two group of data are sampled from Gaussian distribution.

For two group of data X and Y (#X = #Y = n), we ranked these two groups as x_i, y_i (i=1..n), x_bar, y_bar for the average of these two group of data. Then the Spearman correlation coefficient can be formulated as:

$$\rho = \frac{\Sigma_{i=1}^{n}(x_{i}-\bar{x})(y_{i}-\bar{y})}{\sqrt{\Sigma_{i=1}^{n}(x_{i}-\bar{x})^{2}\Sigma_{i=1}^{n}(x_{i}-\bar{x})^{2}}} (1)$$

X_i = {0.8, 18, 1.2, 2.3, 2.3}

Then we ranked X_i with ascending order:

X_i = {0.8, 1.2, 2.3, 2.3, 18}

rank = {1, 2, 3, 4, 5}

However we have the same value 2.3 here, so we get the average,

rank_prime = {1, 2, 3.5, 3.5, 5}.

And we have,

Y_i = {7, 0, 27, 50, 28}

after ranking ascendingly,

Y_i = {0, 7, 27, 28, 50}

rank_y = {1, 2, 3, 4, 5}

Then we can calculate for these two groups of numbers,

$$\rho = \frac{\Sigma_{i=1}^{5}(x_{i}-4.92)(y_{i}-22.4)}{\sqrt{\Sigma_{i=1}^{5}(x_{i}-4.92)^{2}\Sigma_{i=1}^{5}(x_{i}-22.4)^{2}}} = -0.0513 $$

, which is the Spearman rank correlation of two groups of number here.

Note 1
We can also calculate the result with Python.
scipy.stats.spearmanr(a, b=None, axis=0, nan_policy=’propagate’, alternative=’two-sided’)

Note 2
When all of the ranks are integers, the distance can be calculated as:
$$\rho = 1 - \frac{\Sigma_{i=1}^{n}d_{i}^{2}}{n(n^2-1)}(2)$$

,where d_i = rank_x_i - rank_y_i

Note 3
sometimes when ranks are not integer, for the consideration of complexity of calculation, we also use the (2), which might leads to differet result of calculation. Need to check source code for that. $\rho$ supposed to be [-1, 1].
